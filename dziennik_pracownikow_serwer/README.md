#
 > clone code
```
git clone https://github.com/jakub-malec826/FastAPI_project_python_WSZ.git
```

 > create venv
```
python -m venv venv
```

 > run venv: linux | macos
```
source venv/bin/activate
```
> run venv: windows
```
./venv/bin/activate
```

 > install libraries
```
pip install -r requirements.txt
```

 > run server
```
uvicorn main:mainRouter --reload --port 8000
```
